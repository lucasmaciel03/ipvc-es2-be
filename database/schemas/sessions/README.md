# Sessions Schema

Este schema é responsável pela gestão das sessões dos usuários no sistema.

Funcionalidades principais:
- Controle de sessões ativas
- Gestão de tokens de acesso
- Rastreamento de tempo de sessão
- Limite de sessões simultâneas

O schema garante:
- Segurança no acesso
- Controle de concorrência
- Gestão eficiente de recursos
- Monitoramento de atividade dos usuários
